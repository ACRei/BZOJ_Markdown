# #4314. 倍数？倍数！

内存限制：512 MiB时间限制：20 Sec

## 题目描述

求从0 ~ N &ndash; 1中选出K个互不相同的数，使得这些数的和mod N = 0的方案数。输出这个方案数mod (10 ^ 9 + 7)的值。

## 输入格式

仅一行，两个数N,K。

## 输出格式

仅一行，即答案。

## 样例

### 样例输入

    
    7 4
    

### 样例输出

    
    5
    

## 数据范围与提示

{0, 1, 2, 4}, {0, 3, 5, 6}, {1, 2, 5, 6}, {1, 3, 4, 6}, {2, 3, 4, 5}

100 : N <= 10 ^ 9, K <= 1000
