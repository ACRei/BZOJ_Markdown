# #2058. [Usaco2010 Nov]Cow Photographs

内存限制：64 MiB时间限制：3 Sec

## 题目描述

奶牛的图片 
Farmer John希望给他的N(1

## 输入格式

第1行:一个单独的数N
第2到n+1行:第i+1行上的数表示站在第i的位置上的奶牛的编号(即c_i).

## 输出格式

一个整数,表示是奶牛的序列变为一个合法的序列的最小花费时间.
