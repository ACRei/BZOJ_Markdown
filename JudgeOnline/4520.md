# #4520. [Cqoi2016]K远点对

内存限制：512 MiB时间限制：30 Sec

## 题目描述

已知平面内 N 个点的坐标，求欧氏距离下的第 K 远点对。

## 输入格式

输入文件第一行为用空格隔开的两个整数 N， K。接下来 N 行，每行两个整数 X,Y，表示一个点

的坐标。1 < =  N < =  100000， 1 < =  K < =  100， K < =  N*(N&minus;1)/2 ， 0 < =  X, Y < 2^31。

## 输出格式

输出文件第一行为一个整数，表示第 K 远点对的距离的平方（一定是个整数）。

## 样例

### 样例输入

    
    10 5
    0 0
    0 1
    1 0
    1 1
    2 0
    2 1
    1 2
    0 2
    3 0
    3 1
    

### 样例输出

    
    9
