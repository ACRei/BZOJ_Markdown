# #2471. Count

内存限制：128 MiB时间限制：10 Sec

## 题目描述

给定正整数N和M，将1到10^N-1的数字写在字符串上，例如N=2，字符串则是
&ldquo;123456789101112&bdquo;9899&rdquo;。问：M在字符串中出现了多少次？ 
    N < = 15，M< = 10^6.
