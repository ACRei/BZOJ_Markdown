# #2189. 三维最近点对

内存限制：259 MiB时间限制：50 Sec

## 题目描述

给定三维世界的N个点，输出Min{Dist(i,j) | i不等于j }
Dist( I,j ) = Sqr( X[i]-X[j] ) + Sqr( Y[i]-Y[j] ) + Sqr( Z[i]-Z[j] ) 


## 输入格式

N
X1 Y1 Z1
….
Xn Yn Zn

坐标均为不超过10^7的正整数。

## 输出格式

最小的距离。



## 样例

### 样例输入

    
    2
    1 1 1
    1 1 1
    
    

### 样例输出

    
    0
