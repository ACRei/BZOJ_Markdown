# #4291. [PA2015]Kieszonkowe

内存限制：256 MiB时间限制：10 Sec

## 题目描述

给定n个数，请从中选出若干个数，使得总和为偶数，请最大化这个总和。

## 输入格式

第一行包含一个正整数n(1<=n<=1000000)。

第二行包含n个正整数a_1,a_2,...,a_n(1<=a_i<=1000)。

## 输出格式

输出一个正整数，即最大的总和，若无解则输出NIESTETY。

## 样例

### 样例输入

    
    5
    2 4 3 3 3
    

### 样例输出

    
    12
