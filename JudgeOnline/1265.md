## 输入格式

输入文件的第一行有4个正整数：a, b, c和m；而第二行则仅含一个正整数k。它们的含义见上文描述。

## 输出格式

你的程序将向输出文件输出两行，第一行是一个整数P而第二行是一个整数Q。

## 样例

### 样例输入

    
    0 1 1 10
    10000
    
    

### 样例输出

    
    89
    113
    
    

## 数据范围与提示

0 < = a   < =  b  < =  c< = 100, 1 < = m < = 3  000, 1 < = k < = 10^ 6 000
