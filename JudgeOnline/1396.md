# #1396. 识别子串

内存限制：162 MiB时间限制：10 Sec

## 题目描述

![](images/1396.jpg)

## 输入格式

一行,一个由小写字母组成的字符串S,长度不超过10^5

## 输出格式

L行,每行一个整数,第i行的数据表示关于S的第i个元素的最短识别子串有多长.

## 样例

### 样例输入

    
    agoodcookcooksgoodfood
    
    

### 样例输出

    
    1
    2
    3
    3
    2

    3
    3
    2
    2
    3
    3

    1

    4
