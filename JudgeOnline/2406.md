# #2406. 矩阵

内存限制：128 MiB时间限制：10 Sec

## 题目描述

![](upload/201107/2(1).jpg)

## 输入格式

第一行两个数n、m，表示矩阵的大小。

接下来n行，每行m列，描述矩阵A。

最后一行两个数L，R。

## 输出格式

第一行，输出最小的答案；

## 样例

### 样例输入

    
    2 2
    
    0 1
    
    2 1
    
    0 1
    
    
    

### 样例输出

    
    1
    
    

## 数据范围与提示

对于100%的数据满足N,M<=200,0<=L<=R<=1000,0<=Aij<=1000
