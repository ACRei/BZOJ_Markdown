# #1372. [Baltic2003]Lamps

内存限制：64 MiB时间限制：20 Sec

## 题目描述

n个灯围成一个圈。 每个灯可以打开或者关闭。 对于第i个灯，如果第i+1个灯是打开的，则它在下一秒该变其状态。对于第 n个灯的话，则看第 1个灯的状态 给出这些灯最开始的状态，求第M秒后的状态.

## 输入格式

第一行给出 N,M。（0

## 输出格式

输出 N 个数，代表M秒后每个灯的状态

## 样例

### 样例输入

    
    3 1
    0
    0
    1
    

### 样例输出

    
    0
    1
    1
