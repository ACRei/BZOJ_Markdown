# #1356. [Baltic2009]Rectangle

内存限制：64 MiB时间限制：10 Sec

## 题目描述

给出n个点，要你从这些点中找出四个点来组成一个矩形，面积最大.

## 输入格式

第一行给出N.4 &le; n &le; 1,500. 下面N行给出这些点的坐标，其值在 [10^8,10^8]

## 输出格式

最大的矩形面积

## 样例

### 样例输入

    
    8
    -2 3
    -2 -1
    0 3
    0 -1
    1 -1
    2 1
    -3 1
    -2 1
    

### 样例输出

    
    10
