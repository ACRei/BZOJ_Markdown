# #2456. mode

内存限制：1 MiB时间限制：1 Sec

## 题目描述

给你一个n个数的数列，其中某个数出现了超过n div 2次即众数，请你找出那个数。

## 输入格式

第1行一个正整数n。
第2行n个正整数用空格隔开。

## 输出格式

    一行一个正整数表示那个众数。

## 样例

### 样例输入

    
    5
    3 2 3 1 3
    
    

### 样例输出

    
    3
    
    

## 数据范围与提示

100%的数据，n<=500000，数列中每个数<=maxlongint。
