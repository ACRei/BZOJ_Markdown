# #3412. [Usaco2009 Dec]Music Notes乐谱

内存限制：128 MiB时间限制：3 Sec

## 题目描述

![](upload/201401/af(7).jpg)

## 输入格式

    第1行：两个整数N，Q.

    第2到N+1行：第i+l行只有一个整数Bi．

    第N+2到N+Q+I行：第N+i+l行只有一个整数Ti．

## 输出格式

    第1到Q行：对与每个询问，在词问的时间内，奶牛敲击的是哪个音阶？

## 样例

### 样例输入

    
    3 5
    2
    1
    3
    2
    3
    4
    0
    1
    

### 样例输出

    
    2
    3
    3
    1
    1
