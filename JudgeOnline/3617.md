# #3617. dzy loves game

内存限制：512 MiB时间限制：100 Sec

## 题目描述

![](upload/201405/aa.jpg)

## 输入格式

第一行两个整数N,K.
接下来每行N个整数描述Boss的属性。
最后一行N个整数表示Ai

## 输出格式

仅一行，表示最少损失血量，保留三位小数

## 样例

### 样例输入

    
    1 2
    1 1
    0
    
    
    

### 样例输出

    
    1.414
    
    
    

## 数据范围与提示


N<=50000,K<=3
保证A1=0，其它所有Ai>0
属性值小于等于100，完全随机。
仅有三个测试点
