# #1429. 方程的解

内存限制：162 MiB时间限制：10 Sec

## 题目描述

令F(x)=Sigma(i),1<=i<=x. 

有一个不定方程f(x)+f(y)+f(z)+f(w)=N(0<=N<=10^12,x,y,z,w为自然数) 

请统计方程的个数

## 输入格式

一个正数N

## 输出格式

一个数表示解的个数

## 样例

### 样例输入

    
    input 1
    2
    
    input 2
    4
    

### 样例输出

    
    output 1
    6
    
    output 2
    13
