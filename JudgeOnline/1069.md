# #1069. [SCOI2007]最大土地面积

内存限制：128 MiB时间限制：1 Sec

## 题目描述

　　在某块平面土地上有N个点，你可以选择其中的任意四个点，将这片土地围起来，当然，你希望这四个点围成
的多边形面积最大。

## 输入格式

　　第1行一个正整数N，接下来N行，每行2个数x,y，表示该点的横坐标和纵坐标。

## 输出格式

　　最大的多边形面积，答案精确到小数点后3位。

## 样例

### 样例输入

    
    5
    0 0
    1 0
    1 1
    0 1
    0.5 0.5
    

### 样例输出

    
    1.000
    

## 数据范围与提示

数据范围 n<=2000, |x|,|y|<=100000
