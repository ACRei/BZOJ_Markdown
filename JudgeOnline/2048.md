# #2048. [2009国家集训队]书堆

内存限制：259 MiB时间限制：10 Sec

## 题目描述

![](images/2048.jpg)

## 输入格式

第一行正整数 N M

## 输出格式

	一行（有换行符），L，表示水平延伸最远的整数距离 (不大于答案的最大整数)

## 样例

### 样例输入

    
    样例
    #1
    Input:	1 100
    Output:	49
    
    #2
    Input:	2 100
    Output:	74
    
    
    

### 样例输出

    
     N
