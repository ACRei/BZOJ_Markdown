# #4311. 向量

内存限制：512 MiB时间限制：20 Sec

## 题目描述

你要维护一个向量集合，支持以下操作：

1.插入一个向量(x,y)

2.删除插入的第i个向量

3.查询当前集合与(x,y)点积的最大值是多少。如果当前是空集输出0

## 输入格式

第一行输入一个整数n，表示操作个数

接下来n行，每行先是一个整数t表示类型，如果t=1，输入向量

(x,y)；如果t=2，输入id表示删除第id个向量；否则输入(x,y)，查询

与向量(x,y)点积最大值是多少。

保证一个向量只会被删除一次，不会删没有插入过的向量

## 输出格式

对于每条t=3的询问，输出一个答案

## 样例

### 样例输入

    
    5
    1 3 3
    1 1 4
    3 3 3
    2 1
    3 3 3
    

### 样例输出

    
    18
    15
    

## 数据范围与提示

n<=200000 1<=x,y<=10^6
